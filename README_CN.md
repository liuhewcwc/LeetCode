### 简介.

- LeetCode 答案解析集合
- 启动日期：2019年06月22号
- 主语言：Java

---

### 如何做贡献？

- :star: Star 和 Fork 来激励作者持续更新题型和讲解
- :hammer:分享你的代码 （需附有详细解释或者在可能迷惑的代码块附有注释）
- :tada: 在社交圈和软件分享，让更多人知道
- :bulb: Pull request 和 Issues 来更好地维护代码

---

### 如何分享代码？

- 在每个文件的 `code` 板块下
- 清楚地标示所使用的语言
- 附上清晰的解释或批注
- Pull request 合并到 `master`
